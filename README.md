# Build a Discount Calculator

Build a Discount Calculator that can apply different discount strategies to products.  The system will determine the best price for a customer based on multiple discount rules. 

## Technologies

Uses Python3 and Object-Oriented Programming.

## Deployment

Copy `main.py` into the text editor of your choice.  If using terminal, use the command `python3 main.py` and press `Enter` to run.
